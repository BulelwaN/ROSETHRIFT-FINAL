# ROSETHRIFT 

## Description

Rosethrift is an online second-hand clothing store developed using PHP, MySQL, HTML, CSS and JavaScript. The website allows users to browse, buy and sell pre-owned fashion items while promoting sustainable shopping.

---

## Features

### Customer Features

* Register an account
* Login and logout
* Browse products
* View product details
* Add products to cart
* Add products to wishlist
* Checkout products
* View purchase history
* Contact the store
* Sell clothing items

### Admin Features

* Admin login
* View dashboard
* Manage users
* Manage products
* Review sell requests

## How to Run the Project

1. Download or clone the project.

2. Place the project folder inside:

For WAMP:

```text
www/
```

3. Start Apache and MySQL.

4. Open phpMyAdmin.

5. Create a database and import the SQL file.

6. Check the database connection inside `DBConn.php`.

7. Open your browser and run:

```text
http://localhost/ROSETHRIFT-FINAL-main/
```

The purpose of Rosethrift is to encourage sustainable fashion by allowing users to buy and sell second-hand clothing online.
